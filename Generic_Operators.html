procedure

(append a b) → appendable?

  a : appendable?
  b : appendable?
A function taking two arguments that composes them using the implemention of gen:appendable for the type of the arguments being composed.

procedure

(~ v ...) → appendable?

  v : appendable?
Append the provided values together, using the canonical append operation on the data based on its type. The special value ID serves as the generic identity value when the type of the operands is not known. In particular, this value is the result when no operands are provided.
